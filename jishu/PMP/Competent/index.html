何为称职的项目经理

发表于 2020-09-20 分类于 项目管理

摘要：项目经理的能力要求

-

项目经理的职责

  1. 项目经理和指挥都需要为团队的成果负责
  2. 领导团队实现项目目标。

项目经理的定位

  1. 项目经理还充项目问题解决的子项目发起人
  2. 团队成员与其他相关方之间的沟通者

项目经理的工作

  1. 创建、维护和遵循沟通计划和进度计划以及其他项目过程策划能力
  2. 不断地以可预见的方式进行沟通
  3. 寻求了解项目相关方的沟通需求
  4. 对项目资源的需求（有效的组织成员前往现场支持项目）
  5. 可交付成果的接受或发布（作为公司产品-客户的中间桥梁）
  6. 组织项目阶段性成果发布（面向客户）

项目经理的能力

  1. 汇报能力
  2. 文件编纂能力
  3. 对内对外有效的沟通能力
  4. 系统性思维能力
  5. 激励和带领团队的能力
  6. 项目整合管理的能力
  7. 对于项目具体工作的管理把控能力

CIDC项目特有

  1. 软件能力（用于审核各项交付文件及过程文件）
  2. 硬件能力（用于审核各项交付文件及过程文件）
  3. 协调生产发货能力（用于审核各项交付文件及过程文件）
  4. 算法识别能力（用于审核各项交付文件及过程文件）
  5. 综合能力（用于识别单一问题对整个项目哪些模块有影响，组织专项人员处理问题）
